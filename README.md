Description
===========

Install opencv.

Requirements
============

## Platform:

* Ubuntu
* Windows
* CentOS

## Coobooks:

* apt
* chocolatey
* yum
* python

Attributes
==========

See `attributes/default.rb` for default values.

Recipes
=======

default
-------

Launch the "install" recipe.

Usage
=====

Include `recipe[opencv]` on systems where you want to install optipng.
